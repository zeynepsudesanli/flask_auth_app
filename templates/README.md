# Flask Authentication App

This project is a simple web application built with **Flask**. 
It includes a basic homepage and a login system.

## Features
- Home page (`index.html`)
- Login page (`login.html`)
- Simple form-based authentication
- Flask templates for HTML rendering

## Project Structure
flask_auth_app/
- app.py
- templates/
  - index.html
  - login.html
- (optional) static/  # for CSS, JS, images

## Technologies Used
- Python
- Flask
- HTML

## How to Run

1. Open terminal/command prompt in the project folder:
2. Run the Flask application:
3. Open a browser and visit:
- [http://127.0.0.1:5000/](http://127.0.0.1:5000/) → Home page
- [http://127.0.0.1:5000/login](http://127.0.0.1:5000/login) → Login page
